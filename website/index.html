Build the APK

# Requirements: Android Studio + ARM64 device (Android 7+)

# 1. Open in Android Studio and connect your device
# 2. Run → "Run 'app'"

# Or build via GitHub Actions (auto-builds on push to main)
# Download the APK from the Actions → Artifacts tab

# Local build (requires Android SDK):
chmod +x gradlew
echo "sdk.dir=$ANDROID_HOME" > local.properties
./gradlew assembleDebug
# Output: app/build/outputs/apk/debug/app-debug.apk
